# Design notes

Historical design specs and plans captured while building SpeakWell. These are
kept for context on *why* things are built the way they are — they are **not**
user documentation and may lag behind the current code.

For how to set up and use SpeakWell, see the [main README](../../README.md).

- `specs/` — design specs for individual features.
- `plans/` — implementation plans.
